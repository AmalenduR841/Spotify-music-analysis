## Project Overview
End-to-end data analytics project analyzing 32,833 
Spotify tracks across 6 genres from 1957 to 2020.

## Tools Used
- Python (Spotify API + Pandas)
- Power BI Desktop
- DAX Measures

## Dataset Details
| Feature | Value |
|---|---|
| Total Tracks | 32,833 |
| Total Artists | 10,686 |
| Genres | 6 |
| Years Covered | 1957 - 2020 |
| Total Columns | 23 |

## Key Insights
- EDM is fastest genre at 125.8 BPM
- Pop dominates with 18.41% of all tracks
- Average popularity score: 42.48
- Average energy level: 0.70
- Music data spans 64 years

## Audio Features Analyzed
- Danceability — how suitable for dancing
- Energy — intensity and activity level
- Valence — musical positivity
- Acousticness — acoustic sound level
- Speechiness — spoken words amount
- Liveness — live recording detection
- Tempo — speed in BPM

## Dashboard Features
- Genre slicer for filtering
- Year range selector
- KPI cards
- Top artists bar chart
- Genre distribution donut
- Popularity trend line chart
- Audio features radar chart
- Tempo treemap by genre

## How to Use
